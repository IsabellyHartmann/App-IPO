# App-IPO
Front-end em React para o sistema de gestão de um Centro de Inspeções Periódicas Obrigatórias (IPO).

# Objetivo geral
O objetivo é criar uma interface web moderna e funcional que consome essa API e permita a gestão completa
desses três módulos.

Desenvolver uma aplicação React que permita:

- Consultar (GET)
- Adicionar (POST)
- Atualizar (PUT)
- Eliminar (DELETE)
os registos de Clientes, Veículos e Inspeções através da API em NodeJS.

# Começar a APP
Colocar no terminal: 
Para entrar na pasta
cd ipo
npm run dev
